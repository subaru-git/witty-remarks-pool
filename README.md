# Witty Remarks Pool

本を読んで感銘を受けたところを共有する場所です。本に限らず、名言などもあります。
インプットのモチベーションとして用意しました。

サイトはここ

[![icon](public/favicons/favicon-32x32.png)](https://witty-remarks-pool.vercel.app/)

## Add your remarks

`data/data.json` に追加してください。

## Development

```bash
yarn && yarn dev
```
